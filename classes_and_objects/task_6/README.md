# Задача «Мой лифт в многоэтажке»

Вы — **умный лифт** в доме!  
Можете **ехать вверх, вниз, открывать и закрывать двери**,  
и **говорите**, на каком этаже сейчас!  
Но **нельзя ехать с открытыми дверями**!

### Класс: `Elevator`

```java
public class Elevator {
    int currentFloor;     // текущий этаж (от 1 до 10)
    int totalFloors;      // сколько этажей в доме (например, 10)
    boolean doorsOpen;    // открыты ли двери?
}
```

### Методы (команды лифту):

1. **`goUp()`** — ехать вверх  
   → Только если **двери закрыты**  
   → `currentFloor += 1` (но не выше `totalFloors`)  
   → Вывод: `"Лифт едет вверх... Прибыл на 5 этаж!"`

2. **`goDown()`** — ехать вниз  
   → Только если **двери закрыты**  
   → `currentFloor -= 1` (но не ниже 1)  
   → Вывод: `"Лифт едет вниз... Прибыл на 3 этаж!"`

3. **`openDoors()`** — открыть двери  
   → `doorsOpen = true`  
   → Вывод: `"Двери открыты. Заходите!"`

4. **`closeDoors()`** — закрыть двери  
   → `doorsOpen = false`  
   → Вывод: `"Двери закрыты. Едем!"`

5. **`status()`** — показать панель лифта  
   → Вывод:
   ```
   ЛИФТ
   Этаж: 4 из 10
   Двери: открыты
   ```

---

### В `main` — катаемся на лифте!

```java
public static void main(String[] args) {
    Elevator lift = new Elevator();
    lift.currentFloor = 1;
    lift.totalFloors = 10;
    lift.doorsOpen = true;

    lift.status();
    lift.closeDoors();
    lift.goUp();
    lift.goUp();
    lift.openDoors();
    lift.status();
    lift.goUp();        // не поедет!
    lift.closeDoors();
    lift.goUp();
    lift.status();
}
```
